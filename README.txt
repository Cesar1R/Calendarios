El objetivo principal de este proyecto es obtener los eventos de Google Calendar a partir de su API para realizar reportes orientados al analisis de datos, en el caso  de prueba, se utilizaron las citas de un centro de salud, por lo que ademas de los recursos para  reporte anual y mensual, existen codigos que hacen automatizaciones de menor importancia, como enviar las citas diarias a los interesados. 

Se utilizaran el analisis de datos para poder coordinar las campa~nas publicitarias y mantener una cantidad deseada de citas agendadas. 


ADVERTENCIA !!!
En todos los codigos se eliminaron las credenciales que requiere la API y algunos otros datos, por cuestioes de privacidad, por lo que al descargar o copiar los codigos es necesario rellenar estos campos vacios, usualmente estan se~nalados con una fila de guines -----------
