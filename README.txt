El objetivo principal de este proyecto es obtener los eventos de Google Calendar a partir de su API para realizar reportes orientados al analisis de datos, en el caso  de prueba, se utilizaron las citas de un centro de salud, por lo que ademas de los recursos para  reporte anual y mensual, existen codigos que hacen automatizaciones de menor importancia, como enviar las citas diarias a los interesados.
